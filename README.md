# POS Management System

A modern Point of Sale (POS) system built with React and Vite for managing sales, tracking inventory, and viewing business analytics. This comprehensive retail solution combines point-of-sale functionality with real-time inventory management and detailed business intelligence dashboards.

## Features

### Sales & Transactions

- **Sales Recording** - Add items to cart and complete sales transactions
- **Real-time Cart Management** - Dynamic cart with quantity controls
- **Transaction History** - View and manage all past transactions
- **Receipt Generation** - Detailed transaction breakdown with item-level details

### Inventory Management

- **Product Catalog** - Manage 50+ products across 5 categories
- **Real-time Stock Tracking** - Automatic inventory deduction on sales
- **Low Stock Indicators** - Visual alerts for items running low
- **Category Organization** - Stationary, IT Gadgets, Snacks, Consumer Products, Medicines

### Analytics & Reporting

- **Sales Dashboard** - Comprehensive overview with key metrics
- **Sales Trends** - Visualize daily and monthly sales data with interactive charts
- **Category Performance** - Track sales performance across product categories
- **Top Best-Selling Items** - Identify your top 5 performing products
- **Period-based Reporting** - Filter data by Today/Week/Month/All Time
- **Revenue Tracking** - Monitor total revenue and transaction counts

### User Experience

- **Product Search & Filter** - Filter products by category
- **Responsive Design** - Works seamlessly across devices
- **Toast Notifications** - Real-time feedback for user actions
- **Accessible UI** - Built with Radix UI for accessibility standards
- **Date Selection** - Choose transaction dates for backdating sales

## Tech Stack

### Frontend Framework & Build

- **React 19** - Modern UI framework with hooks
- **Vite 7** - Fast build tool with HMR
- **React Router DOM 7** - Client-side routing

### UI & Styling

- **Tailwind CSS 4** - Utility-first CSS framework
- **Radix UI** - Accessible component library
- **Lucide React** - Icon library with 560+ icons
- **Shadcn UI** - Custom UI component system
- **Poppins Font** - Google Fonts integration

### Data Visualization

- **Recharts 3** - Interactive charts (LineChart, BarChart)

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository

```bash
git clone <repository-url>
cd project-one
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

### Available Scripts

- `npm run dev` - Start development server with hot reload
- `npm run build` - Build for production
- `npm run lint` - Run ESLint checks
- `npm run preview` - Preview production build
